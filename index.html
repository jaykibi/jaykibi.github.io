Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

puts "Enter degrees in Celcius: "
n = gets.chomp.to_i

def conversionCtoF(n)
    return n * 1.8 + 32
end

puts "The temperature is #{conversionCtoF(n).to_i} Fahrenheit"

Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

puts "How many items do you want to see?"
user_input = gets.chomp.to_i

def zero(x)
    x == 0
end

def modulo3(x)
    x % 3 == 0
end

def modulo5(x)
    x % 5 == 0 
end

def moduloboth(x)
    x % 3 == 0 && x % 5 == 0 
end

n = 0 
while n != user_input
    if zero(n) == true
        puts n
    elsif moduloboth(n) == true
        puts "FooBar"
    elsif modulo3(n) == true
        puts "Foo"
    elsif modulo5(n) == true
        puts "Bar"    
    else
        puts n 
    end
    n = n + 1 
end
